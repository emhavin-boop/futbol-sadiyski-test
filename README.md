# Футболен тест по правила и съдийство 2026

Интерактивен тест с въпроси за футболни съдии (по PDF "Въпроси - Jan 2026 No duplicates").

- 20 случайни въпроса на тест  
- Опции за време: без лимит / 15 мин / 20 мин  
- Показва верния отговор след избор  
- Процент успеваемост накрая + преглед на отговорите

## Как да ползваш теста

1. Отиди на:  
   [https://emhavin-boop.github.io/futbol-sadiyski-test/quiz.html](https://emhavin-boop.github.io/futbol-sadiyski-test/quiz.html)  
   (ако Pages не е включен – виж по-долу)

2. Или свали файла quiz.html и го отвори в браузър (работи офлайн)

## За да го инсталираш като апликация на телефона

- Отвори линка в Chrome / Safari  
- Натисни **Споделяне** → **Добави към началния екран**  
→ ще се появи икона като нормално приложение
